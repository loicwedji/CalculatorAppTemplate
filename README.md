Calculator App Workshop

![CSC Club Logo](../logo_black.png)

This repository contains a simple Java Swing calculator app template used
during a hands-on workshop focused on learning Git, GitHub, and
collaborative development.

How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/loicwedji/CalculatorAppTemplate.git
cd CalculatorAppTemplate
```

2. Install what you need:

- A Java Development Kit (JDK), such as JDK 17 or newer
- If you are using VS Code, install the `Extension Pack for Java`
- If you are using IntelliJ IDEA, use IntelliJ IDEA Community or Ultimate
  with the project JDK set up

3. Open the project in your editor:

- In VS Code, open the folder and make sure VS Code detects your JDK
- In IntelliJ, open the folder as a project and confirm the Project SDK is
  set to your installed JDK

4. Run the app:

- In VS Code, open `CalculatorAppTemplate.java` and click `Run` above the
  `main` method, or run it from the Run and Debug panel
- In IntelliJ, open `CalculatorAppTemplate.java` and run the
  `main` method
- From the terminal, you can also run:

```bash
javac CalculatorAppTemplate.java
java CalculatorAppTemplate
```

Note: if `java` or `javac` says a runtime cannot be found, install a JDK
first and restart your editor or terminal.
